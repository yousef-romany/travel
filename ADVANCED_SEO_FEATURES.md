## 💡 **How to Use New Features**

### **On Program Pages:**

```tsx
import ProgramFAQSchemaEnhanced from '@/components/seo/ProgramFAQSchemaEnhanced';
import AggregateOfferSchema from '@/components/seo/AggregateOfferSchema';
import AdvancedMetaTags from '@/components/seo/AdvancedMetaTags';

// In your page component:
<ProgramFAQSchemaEnhanced
  programTitle={program.title}
  programLocation={program.Location}
  duration={program.duration}
  price={program.price}
  includes={program.includes}
  excludes={program.excludes}
/>

<AggregateOfferSchema
  productName={program.title}
  offers={[
    { name: "Standard Package", price: program.price },
    { name: "Deluxe Package", price: program.price * 1.3 },
    { name: "Premium Package", price: program.price * 1.6 },
  ]}
  image={program.images[0]}
  url={`/programs/${program.documentId}`}
/>

<AdvancedMetaTags
  title={program.title}
  description={program.description}
  image={program.images[0]}
  url={`/programs/${program.documentId}`}
  price={program.price}
  type="product"
/>
```

### **On Guide/Article Pages:**

```tsx
import HowToSchema from "@/components/seo/HowToSchema";

<HowToSchema
  name="How to Plan Your Egypt Trip"
  description="Step-by-step guide to planning the perfect Egyptian vacation"
  totalTime="PT2H"
  estimatedCost="2000"
  steps={[
    {
      name: "Choose Your Destinations",
      text: "Select from Cairo, Luxor, Aswan, or Red Sea resorts based on your interests.",
    },
    {
      name: "Book Your Tours",
      text: "Reserve tours and activities in advance for better prices and availability.",
    },
    // ... more steps
  ]}
/>;
```

### **Internal Linking:**

